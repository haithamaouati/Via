# Via browser Userscripts

### P2R
Pull-to- refresh

A script that provides a pull-down-to-refresh feature for browsers that do not support this functionality, such as the Via browser.

#### [Install](https://raw.githubusercontent.com/haithamaouati/Via/refs/heads/main/P2R.js)
Import script from URL:
```
https://raw.githubusercontent.com/haithamaouati/Via/refs/heads/main/P2R.js
```
